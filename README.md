# 🛒 Supermarket Sales Dashboard

A Power BI dashboard analyzing supermarket sales 
performance across countries, categories, and time.

## 🔍 Business Questions Answered
- Which countries generate the most revenue?
- What is the return rate and why are customers returning?
- Who are the top 5 customers by total spend?
- How did sales trend throughout 2023?

## 💡 Key Insights
- Total Sales: $3M | Average Order Value: $623
- Return Rate: 10% — "Changed mind" is the #1 reason
- Sales dropped 12.14% between January and August 2023
- Gaming products dominate top price rankings

## 📋 Recommendation
Focus retention efforts on reducing impulse returns 
by improving product descriptions and return policies.

## 🛠️ Tools Used
- Power BI (with Smart Narrative)
- CSV Datasets (Customers, Orders, Products, Returns)

## 📁 Files
- `Dashboard.pbix` - Power BI file
- `Customers.csv`, `Orders.csv`, `Products.csv`, 
  `Returns.csv` - Raw datasets
